# HoTTClient
Arduino library to read telemetry data from Graupner HoTT-sensors
Confirmed sensor firmware version: 1.46 and 2.02.

### Required hardware:
* Arduino Pro Mini 328, 16Mhz, 5V
* 1x resistor 1k5 to 4k7 Ohm (SMD 0805 fits perfect)
